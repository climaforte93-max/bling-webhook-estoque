# Bling Webhook - Notificação de Estoque Baixo

Aplicação que recebe webhooks do Bling sobre alterações de estoque e envia notificações via WhatsApp usando WebZap.

## 📋 Arquivos do Projeto

| Arquivo | Descrição |
|---------|-----------|
| `webhook.js` | Lógica principal do webhook |
| `index.js` | Entry point do servidor |
| `package.json` | Dependências do projeto |
| `vercel.json` | Configuração para Vercel |
| `.env.example` | Exemplo de variáveis de ambiente |
| `.gitignore` | Arquivos ignorados pelo Git |
| `README.md` | Esta documentação |
